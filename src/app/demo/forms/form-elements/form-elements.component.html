Basic Form Elements

Input field

<mat-input>

Input fields with float labels greatly improve the usability of Forms. The user focuses the input and does not lose the context of the field.

Example:

<mat-form-field>
<input matInput>
</mat-form-field>

place location_city local_airport
{{ characterCountHintExample.value.length }} / 100

API Summary

Name Type Description Default Value
align "start" | "end" The alignment of the input or textarea. "start"
dividerColor "primary" | "accent" | "warn" The color of the placeholder and underline. "primary"
floatingPlaceholder boolean Whether the placeholder should float above the input after text is entered. true
hintLabel string The hint text to show on the start end of the mat-form-field null

Checkboxes

<mat-slide-toggle>

Checkboxes allow the user to select multiple options from a set. If you have multiple options appearing in a list, you can preserve space by using checkboxes instead of switches.

Example:
